# SQL Portfolio — Northwind Database

Practice SQL queries using the Northwind database in PostgreSQL.

## Topics Covered
- SELECT, WHERE, ORDER BY, LIMIT
- NULL handling, COALESCE
- GROUP BY, HAVING
- Aggregations: COUNT, SUM, AVG, MIN, MAX
- DATE functions, EXTRACT
- String functions: TRIM, LOWER, CONCAT, REPLACE, CAST
- INNER, LEFT, RIGHT, FULL OUTER JOIN
- Multiple JOINs
- UNION, UNION ALL
- CASE statements
- Subqueries
- CTEs (WITH clause)
- Window functions: RANK, ROW_NUMBER, LAG, LEAD
- DDL: CREATE, INSERT, UPDATE, DELETE, TRUNCATE, DROP
- INDEXES
